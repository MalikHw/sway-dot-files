### How to use:
- install packages
```
sudo pacman -Sy sway swaybg foot waybar wofi fastfetch
```
- clone repo
```
git clone https://github.com/MalikHw/sway-dot-files.git
cd sway-dot-files
```
Install
```
mkdir ~/.config
cp * ~/.config/ -r
```
